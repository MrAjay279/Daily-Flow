# 🌟 Daily Flow – Productivity Web App

A clean and modern **daily productivity web application** that helps you manage tasks, track habits, visualize progress, and calculate BMI — all in one place.

---

## 🚀 Features

### ✅ Task Management

* Add daily tasks with optional time
* Mark tasks as completed
* Delete tasks easily

### 🔁 Habit Tracking

* Create recurring daily habits
* Track completion streaks
* Reset habits anytime

### 📅 Calendar View

* Monthly calendar interface
* Mark days as:

  * 🟢 Productive (Green)
  * 🔴 Unproductive (Red)
* View tasks for any selected day

### 📊 Statistics Dashboard

* Monthly productivity analysis
* Last 7 days progress chart
* Efficiency percentage tracking

### ⚖️ BMI Calculator

* Calculate BMI instantly
* Get health category + advice

### 🎨 UI Features

* Multiple themes (Orange, Green, Dark)
* Smooth animations & transitions
* Mobile-friendly responsive design
* Floating action button (FAB)
