      About Daily Flow

      Daily Flow

      Version 1.4 • Built for better days

      Daily Flow is a clean and simple daily planner that helps you build consistent habits, manage your tasks, and track your progress every day with joy and intention.

      Key Features

      🔄

      Recurring Habits

      Repeat daily or on specific weekdays

      📅

      One-time Tasks

      Tasks for specific days with optional time

      🎨

      Beautiful Calendar

      Mark days green/red to track consistency

      📊

      Progress Tracking

      Live progress bar and monthly stats

      How to Use

      Adding Tasks

      Type your task and tap the + button.
      Select weekdays = Recurring habit
      No days selected = One-time task

      Calendar

      Tap any day to mark it Green (good day) or Red.
      You can also add tasks directly from calendar view.

      Pro Tips

      • • Tap on task text to edit it
      • • Drag & drop tasks to reorder
      • • Tap the time below a task to set reminder
      • • Use floating button to jump back to Today
      • • Mark calendar days consistently to build streaks
